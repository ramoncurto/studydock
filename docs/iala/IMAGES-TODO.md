## Notes

1. **Color Accuracy**: For lateral marks, color is CRITICAL. Red and green must be clearly distinguishable.

2. **Regional Differences**: Ensure images clearly show which region (A or B) they represent.

3. **Topmark Clarity**: Topmarks are small details that must be clearly visible in images.

4. **Light Diagrams**: Light characteristic diagrams should show timing accurately.

5. **Consistency**: Try to maintain consistent style across similar types of marks.

## Bulk Image Creation Strategy

### For Diagrams and Technical Images:
1. Use vector graphics software (Inkscape, Illustrator)
2. Create templates for each mark type
3. Export as high-quality JPEGs
4. Ensure text is readable at all sizes

### For Photographic Images:
1. Search official maritime sources first
2. Ensure proper licensing
3. Crop and enhance for clarity
4. Add labels if needed for educational purposes

### For Maps:
1. Use OpenStreetMap or similar for base maps
2. Add IALA region overlays
3. Clearly mark boundaries between regions
4. Include legend
